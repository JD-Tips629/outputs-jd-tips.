Arrastar a pasta para o Netlify Drop publica a pagina, mas nao liga o backend real.
Para admin real, login, pedidos VIP e palpites guardados no servidor, publica num hosting Node.js.

2. Como entrar no Admin
   - O cliente nao ve o botao Admin.
   - Clica/toca 5 vezes no logotipo JD-Tips.
   - Escreve a palavra-passe Admin.
   - Depois de entrares, aparecem Admin, Editar e Apagar palpites.
   - Clica em "Sair do Admin" quando terminares.

3. Como o cliente entra no VIP
   - O cliente escolhe Semanal, Mensal ou Premium.
   - Paga por MB WAY ou por IBAN/Multibanco.
   - Preenche o formulario de confirmacao.
   - O app abre WhatsApp para ele enviar o pedido para ti.
   - Tu confirmas no banco/MB WAY se o dinheiro entrou.
   - No Admin, marcas o pedido como Pago.
   - Copias o Codigo VIP e envias ao cliente.
   - O cliente coloca o codigo em "Entrar como VIP" e desbloqueia os palpites VIP.

4. Seguidores, feed e resultados
   - O botao "Seguir JD-Tips" fica visivel na barra lateral.
   - O contador de seguidores tambem aparece no Inicio.
   - Os palpites ativos mostram equipas, horario, mercado e odd.
   - No Admin, cada palpite ativo pode ser marcado como Green ou Red.
   - O valor em euros e opcional: podes marcar Green/Red sem escrever valor.
   - O Historico Green/Red fica publico na secao Resultados.
   - Para notificacoes simples, usa o feed do app: os seguidores abrem o app e veem as novidades.
   - Para notificacoes push reais, sera preciso adicionar Web Push ou integrar Telegram/WhatsApp.

5. Estrutura do banco de dados
   - data/db.json guarda:
     followers: lista de seguidores
     tips: palpites ativos e historico Green/Red
     paymentRequests: pedidos VIP
     history: compatibilidade com ganhos antigos
     wallet: total financeiro registado

6. Testar no teu computador
   - Abre a pasta jd-tips no terminal.
   - Executa:
     npm start
   - Abre:
     http://localhost:4173

7. Publicar online com backend real
   - Usa um hosting que aceite Node.js, por exemplo Render, Railway, Fly.io ou um VPS.
   - Faz upload/importacao da pasta jd-tips completa.
   - Define:
     Build command: vazio ou npm install
     Start command: npm start
   - O servidor vai usar a porta indicada pelo hosting atraves de PORT.
   - O hosting deve fornecer HTTPS para o PWA poder instalar no telemovel.
   - No Render, depois de enviar estes ficheiros para o teu repositorio, clica em Manual Deploy ou espera o auto deploy.
   - Se quiseres manter os dados mesmo depois de novos deploys, usa um disco persistente no Render ou troca data/db.json por uma base de dados externa.

10. Nota sobre pagamentos
   - Esta versao usa confirmacao manual de pagamentos.
   - O app nao confirma automaticamente se o MB WAY/IBAN recebeu dinheiro.
   - Para pagamento automatico com ativacao imediata, sera preciso integrar uma plataforma de pagamentos oficial.
